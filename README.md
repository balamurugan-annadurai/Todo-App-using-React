# 📋 Responsive Todo Web App

## 🌟 Overview

Welcome to the **Responsive Todo Web App** – a sleek and user-friendly application designed to streamline your task management. Built with React, this app provides an smooth interface that adapts seamlessly to any device, ensuring a seamless experience whether you're on your desktop, tablet, or smartphone.

## 🚀 Features

- 📱 **Responsive Design**: Enjoy a consistent experience across all devices.
- ✅ **Easy Task Management**: Add, edit, and delete tasks with a few simple clicks or taps.

## 🛠️ Technologies Used

- ⚛️ **React**: A powerful JavaScript library for building user interfaces.
- 🌐 **HTML5 & CSS3**: The latest web standards for structure and style.
- 🎨 **Bootstrap**: A popular CSS framework for responsive design.
- 📚 **JavaScript (ES6+)**: Modern JavaScript features and syntax.

## 📝 Usage

1. **Add a Task:**
   - Type your task in the input field and press Enter or click the Add button.

2. **Edit a Task:**
   - Click on a task to edit it. Make your changes and press Enter to save.

3. **Delete a Task:**
   - Click the delete icon next to a task to remove it from the list.
